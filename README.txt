Онлайн платформа обучения Python и ML.

Платформа позволяет выбирать курсы, изучать темы, решать задания с использованием code-runnera, получать подтверждение о прохождении курса, использовать форум и блог.

Написана на веб-фреймворке Flask (Python) с использованием архитектурного стиля Rest-API. Работа с базой данных реализована через ORM с использованием синхронного драйвера для СУБД postgresql. Авторизация с помощью сессий. Реализовано шифрование пароля. Рендеринг страниц происходит на стороне сервера с помощью шаблонов. Развертывание приложения с помощью docker-compose.

База данных состоит из 20 + 1 (миграции) таблиц.
