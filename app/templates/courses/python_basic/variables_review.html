переменные

Переменные — это именованные области памяти, которые используются для хранения данных. В Python переменные создаются автоматически при первом присваивании значения. Имя переменной должно начинаться с буквы или символа подчеркивания и может содержать буквы, цифры или символы подчеркивания. В переменной могут хранится значения разных типов. Наша задача в течение научиться работать с различными типами данных. Пока что научимся создавать переменную.

Создание переменной: 

num = 2
name = "john doe"

Здесь мы определили две переменные

  • переменная num хранит значение 2 (число)

  • переменная name хранит значение john doe (строку)

Теперь если мы выведем переменные на экран уже известной командой print, то увидим в консоли значения созданных переменных: 

print(num)
print(name)

Названия переменных должны отражать суть значения, а так же не являться ключевым словом. Ключевые слова: 

                        
                            False      await      else       import     pass
                            None       break      except     in         raise
                            True       class      finally    is         return
                            and        continue   for        lambda     try
                            as         def        from       nonlocal   while
                            assert     del        global     not        with
                            async      elif       if         or         yield

типы данных

Как мы уже узнали, переменнная может хранить значение различных типов. Со всеми типами данных мы разберемся в ходе курса, а в этом блоке научимся обрабатывать переменные и строки. Тем не менее приведем список типов данных в Python:

  • Числовые типы:
    - int — целые числа (например, 42)
    - float — числа с плавающей запятой (например, 3.14)
    - complex — комплексные числа (например, 1 + 2j)

  • Строки:
    - str — последовательности символов (например, "Hello, World!")

  • Булевый тип:
    - bool — логические значения (True, False)

  • Списки:
    - list — упорядоченные изменяемые коллекции (например, [1, 2, 3, "apple"])

  • Кортежи:
    - tuple — упорядоченные неизменяемые коллекции (например, (1, 2, 3))

  • Множества:
    - set — неупорядоченные коллекции уникальных элементов (например, {1, 2, 3})

  • Словари:
    - dict — неупорядоченные коллекции пар "ключ-значение" (например, {"name": "Alice", "age": 30})

  • NoneType:
    - None — специальное значение, обозначающее отсутствие значения или "пустоту"

Существуют еще некоторые типы данных, которые встречаются реже, например байтовые последовательности
