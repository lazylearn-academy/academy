цикл while

Цикл while выполняет блок кода, пока заданное условие истинно (True). Основной синтаксис выглядит следующим образом: 

while условие:
    # Блок кода, который будет выполняться,
    # пока условие истинно

Рассмотрим простой пример, чтобы проиллюстрировать, как работает цикл while. Допустим, мы хотим вывести числа от 1 до 5. 

i = 1
while i <= 5:
    print(i)
    i += 1  # Увеличиваем значение i на 1

В этом примере:

  • Переменная i инициализируется значением 1.

  • Цикл while проверяет условие i <= 5. Если условие истинно, выполняется блок кода внутри цикла.

  • Внутри цикла выводится текущее значение i, а затем i увеличивается на 1.

  • Когда i становится равным 6, условие становится ложным, и цикл заканчивается.

Если не обновлять переменную условия внутри цикла или если условие всегда истинно, это приведет к бесконечному циклу. Например: 

while True:
    print("Этот цикл никогда не остановится.")

]

Чтобы выйти из бесконечного цикла, можно использовать оператор break. Например: 

count = 0
while True:
    count += 1
    print(count)
    if count >= 5:
        break  # Прерываем цикл, когда count достигнет 5

Цикл while полезен в ситуациях, когда количество итераций невозможно заранее определить. Например, это может быть полезно для обработки пользовательского ввода: 

user_input = ''
while user_input.lower() != 'exit':
    user_input = input("Введите 'exit' для выхода: ")
    print("Вы ввели:", user_input)

В этом примере цикл продолжается до тех пор, пока пользователь не введет команду exit.

задания

Задача 1: Сумма чисел от 1 до N Напишите программу, которая запрашивает у пользователя число N и находит сумму всех целых чисел от 1 до N. Используйте цикл while для выполнения этой задачи.

Задача 2: Угадай число Создайте простую игру, в которой программа загадывает случайное число от 1 до 100, а пользователь должен угадать его. Программа должна запрашивать ввод числа у пользователя до тех пор, пока он не угадает загаданное число. При каждом неправильном вводе программа должна сообщать, больше или меньше загаданное число. Используйте цикл while для реализации этой игры.

Задача 3: Факториал числа Напишите программу, которая запрашивает у пользователя неотрицательное целое число N и вычисляет его факториал (N!). Факториал числа N — это произведение всех положительных целых чисел от 1 до N. Реализуйте решение с помощью цикла while.

Задача 4: Показатель степени Создайте программу, которая запрашивает у пользователя два числа: основание и показатель степени. Программа должна вычислить и вывести результат возведения основания в указанный показатель с использованием цикла while. Например, если пользователь ввел 2 и 3, программа должна вычислить 2^3 (что равно 8). Помните, что всякий раз, когда пользователь вводит показатель 0, результат должен быть 1, а для отрицательных показателей следует обрабатывать отдельно.
