первая программа и правила оформления кода

Курс Python (базовый). блок 1, тема 2.

программа

Программа — это последовательность инструкций, написанных на определенном языке программирования, которая выполняет конкретную задачу. Программы могут варьироваться от простейших (например, вывод "Hello, World!") до сложных систем, таких как операционные системы или веб-приложения.

Программа обычно состоит из следующих компонентов:

  • Исходный код: Текст, написанный на языке программирования, который содержит все инструкции для выполнения. Код пишется в текстовом редакторе или интегрированной среде разработки (IDE).

  • Алгоритмы: Четкие шаги, которые описывают логику выполнения задач. Каждый алгоритм может быть реализован в программе с помощью кода.

  • Данные: Входные или выходные значения, которые программа использует во время выполнения. Данные могут храниться в переменных, списках, кортежах и других структурах данных.

  • Функции: Блоки кода, которые выполняют определенные задачи. Функции позволяют организовать код и повторно использовать его.

Со всеми частями программ мы познакомимся в ходе курса, а пока что напишем и разберем нашу первую программу.

практика

Давайте напишем нашу первую программу на Python, состоящую из одной инструкции. Наша цель — вывести на экран фразу "Hello, World!". 

print("Hello, world!")

Объяснение кода:

  • print() — это встроенная функция в Python, которая выводит текст или значения переменных на экран.

  • "Hello, World!" — это строка текста, которую мы хотим вывести.

Теперь давайте запустим программу:

  • Откройте IDE, например, PyCharm или VSCode.

  • Создайте новый файл с расширением .py, например, hello.py.

  • Вставьте код в файл и сохраните изменения.

  • Запустите программу с помощью командной строки: 

    python hello.py

В результате на экране в командной строке должна появиться строка "Hello, world!"

правила оформления кода

Мы написали программу из одной инструкции, но обычно инструкций много. Чтобы программа работала без ошибок и все инструкции выполнялись нужно следовать правилам оформления кода. Также оформлять код правильно нужно, чтобы не запутать в нем себя и товарищей.

Программа обычно состоит из следующих компонентов:

  • Новые инструкции помещаются с начала строки. Например: 

    a = 2
    b = 2
    print(a + b)

  • В некоторых конструкциях необходи отступ 4 пробела или символ табуляции. Во всей программе отступы должны быть выполнены одним из этих способов. Пример программы с отступом: 

    a = 2
    b = 3
    if a > b:
        print(a + b)

  • Регистры в python имеют значение, например функция print не выполнится если допустить ошибку и написать Print.

  • Для лучшей читаемости кода следует добавлять комментарии. Комментарии должны помогать разработчику быстрее и правильнее разобраться в коде. Комментарии, указанные в коде, не выполняются. Бывают однострочные комментарии (# комментарий): 

    # сложение двух чисел в Python
    a = 2
    b = 2
    print(a + b)
    и многострочные ("""комментарий"""): 
    """
      сложение двух чисел
      в Python
    """
    a = 2
    b = 2
    print(a + b)
