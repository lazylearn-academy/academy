числа

Python поддерживает несколько типов чисел:

  • Целые числа (int)
    Целые числа — это числа без дробной части. В Python они могут быть как положительными, так и отрицательными. Также в Python нет ограничения на размер целого числа, помимо исчерпания оперативной памяти. 

    a = 5
    b = -3
    c = 99999999999999999999999999999999  # Большое целое число

  • Вещественные числа (float)
    Вещественные числа — это числа с плавающей запятой (дробные числа). Они представляют собой числа с десятичной частью. 

    x = 3.14
    y = -0.001
    z = 2.0

  • Комплексные числа (complex)
    Комплексные числа состоят из действительной и мнимой частей и обозначаются с помощью буквы j. 

    complex_number = 1 + 2j

операции с числами

Python позволяет выполнять основные арифметические операции:

  • Сложение (+)

  • Вычитание (-)

  • Умножение (*)

  • Деление (/)

  • Целочисленное деление (//)

  • Остаток от деления (%)

  • Возведение в степень (**)

a = 10
b = 3
print(a + b)  # 13
print(a - b)  # 7
print(a * b)  # 30
print(a / b)  # 3.333...
print(a // b) # 3
print(a % b)  # 1
print(a ** b) # 1000

Сложные операции с числами такие как расчет тригономитрических функций можно выполнить с помощью библиотек. Об этом чуть позже в курсе.
