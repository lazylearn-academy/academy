строки

В этой лекции мы углубимся в работу со строками в языке Python, посмотрим, как они хранятся в памяти, а также рассмотрим основные методы работы со строками и как получать пользовательский ввод. Строки в Python — это последовательности символов, используемые для представления текстовой информации. Они могут заключаться в одинарные (' ') или двойные кавычки (" "). Например: 

s1 = 'Привет, мир!'
s2 = "Python - это просто!"

Строки нужно где-то хранить. Рассмотрим как строки хранятся в памяти. Строки в Python хранятся в виде неизменяемых (immutable) объектов. Это означает, что после создания строки её нельзя изменить. Каждое изменение строки приводит к созданию новой строки в памяти. Это к примеру экономит память в случае повторного использования одинаковых строк.

В Python строки являются последовательностью символов и поддерживают индексацию, что позволяет обращаться к отдельным символам по их позициям. Индексация начинается с нуля, поэтому первый символ строки имеет индекс 0, второй — 1, и так далее. Для обращения к символу строки можно использовать квадратные скобки []. Например: 

my_string = "Привет!"
first_character = my_string[0]  # Получаем первый символ: 'П'
last_character = my_string[-1]   # Получаем последний символ: '!'

print(first_character)  # Вывод: П
print(last_character)   # Вывод: !

основные методы работы со строками

Python предоставляет множество встроенных методов для работы со строками. Рассмотрим основные:

  • len(): Вернуть длину строки. 

    s = "Программирование"
    length_s = len(s)  # Вернет 16

  • upper() и lower(): Преобразовать строку в верхний или нижний регистр. 

    s = "Привет"
    s_upper = s.upper()  # Вернет 'ПРИВЕТ'
    s_lower = s_lower()  # Вернет 'привет'

  • strip(): Удалить пробелы в начале и конце строки. 

    s = "  Привет, мир!   "
    s_cleaned = s.strip()  # Вернет 'Привет, мир!'

  • split(): Разделить строку на части по заданному разделителю. Эти части формируют тип данных список, который в отличие от обычной переменной может хранить множество значений. За это свойство список называется коллекцией. О коллекциях вы узнаете совсем скоро, а пока следует запомнить этот важный метод. 

    s = "яблоко, груша, банан"
    fruits = s.split(", ")  # Вернет ['яблоко', 'груша', 'банан']

  • replace(): Заменить одну часть строки другой. 

    s = "Привет, мир!"
    new_s = s.replace("мир", "Python")  # Вернет 'Привет, Python!'

  • find(): Найти позицию первого вхождения подстроки. 

    s = "Привет, мир!"
    position = s.find("мир")  # Вернет 8

конкатенация

Конкатенация — это операция, в результате которой объединяются две или более последовательности (строки, списки и т. д.) в одну. В Python конкатенация строк и списков осуществляется с помощью оператора +. 

s1 = "Привет, "
s2 = "мир!"
concatenaded_string = s1 + s2
print(concatenaded_string)  # Вывод: "Привет, мир!"

пользовательский ввод. конкатенация строк.

Получение пользовательского ввода в Python осуществляется с помощью функции input(). Всё, что вводит пользователь, воспринимается как строка. Это очень важно! Если мы хотим получить от пользователя число, необходимо введенную преобразовать в нужный тип числа, об этом в следующей теме блока. 

name = input("Введите ваше имя: ")
print("Здравствуйте, " + name + "!")
