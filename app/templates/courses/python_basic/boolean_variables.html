булевые переменные

Булевы переменные — это переменные, которые могут принимать только два значения: True (истина) и False (ложь). Они играют ключевую роль в логических операциях, условных выражениях и управлении потоками исполнения в программах. Например мы можем присвоить булевой переменной значение активен ли пользователь и закончени ли некий процесс (пример ниже). Условные выражения мы разберем в следующей теме, пока что запомните что представляют булевы переменные.

В Python булевы переменные создаются просто присваиванием значения True или False. 

is_active = True
is_complete = False

Важно отметить, что True и False в Python являются специальными значениями и пишутся с заглавной буквы.

Булевы переменные могут быть использованы с логическими операциями:

  • AND (и) – возвращает True, если оба операнда истинны.

  • OR (или) – возвращает True, если хотя бы один из операндов истинен.

  • NOT (не) – инвертирует значение (истина становится ложью и наоборот).

Примеры логичеких операций: 

a = True
b = False
print(a and b)  # Вывод: False
print(a or b)   # Вывод: True
print(not a)    # Вывод: False

Python имеет встроенные функции для преобразования значений в булевы типы:

  • bool(0) возвращает False

  • bool(1) возвращает True

  • bool("") возвращает False

  • bool("Hello") возвращает True

  • bool([]) возвращает False

  • bool([1, 2, 3]) возвращает True

Булевы переменные часто используются в различных областях программирования:

  • Проверка условий на уровне бизнес-логики.

  • Управление доступом и аутентификация пользователей.

  • Управление состоянием объектов.

  • Логические тесты и валидация данных.
