if, elif, else

Конструкции if, elif и else являются основными инструментами для управления потоком выполнения программ на Python. Они позволяют выполнять определенные блоки кода в зависимости от выполнения условий. Понимание этих конструкций является основой для написания эффективных и логически последовательных программ.

Конструкция if используется для проверки условия. Если условие истинно (True), то выполняется блок кода, находящийся под ним.

Синтаксис: 

if условие:
    # код выполняется, если условие истинно

Пример: 

temperature = 30
if temperature > 25:
    print("Сегодня жарко!")

В этом примере программа выведет "Сегодня жарко!", так как указанное условие истинно.

Конструкция else позволяет выполнить блок кода, если условие в if ложно (False).

Синтаксис: 

if условие:
    # код, который выполняется, если условие истинно
else:
    # код, который выполняется, если условие ложно

Пример: 

temperature = 20

if temperature > 25:
    print("Сегодня жарко!")
else:
    print("Сегодня прохладно!")

В этом примере программа выведет "Сегодня прохладно!", так как условие во первом if ложно.

Конструкция elif (сокращение от "else if") позволяет проверить несколько условий последовательно. Это полезно, когда необходимо сравнить переменную с несколькими значениями, и выполнить различные действия в зависимости от результата.

Синтаксис: 

if условие1:
    # код выполняется, если условие1 истинно
elif условие2:
    # код выполняется, если условие2 истинно
else:
    # код выполняется, если ни одно из условий не истинно

Пример: 

temperature = 15

if temperature > 25:
    print("Сегодня жарко!")
elif temperature > 10:
    print("Сегодня тепло!")
else:
    print("Сегодня холодно!")

В этом примере программа выведет "Сегодня тепло!", так как первое условие ложно, а второе — истинно.

Можно использовать несколько конструкций elif, чтобы проверить несколько условий подряд: 

score = 85

if score >= 90:
    print("Отлично!")
elif score >= 75:
    print("Хорошо!")
elif score >= 60:
    print("Удовлетворительно!")
else:
    print("Неудовлетворительно!")

В этом случае программа проверяет каждое условие последовательно и выводит соответствующее сообщение.

логические операторы

Вы можете комбинировать условия с помощью логических операторов: and, or, not.

Пример: 

age = 20
has_permission = True

if age >= 18 and has_permission:
    print("Доступ разрешен!")
else:
    print("Доступ запрещен!")

В этом примере доступ будет разрешен только если оба условия истинны.

вложенные конструкции if

Вы можете вкладывать конструкции if друг в друга, чтобы проверять дополнительные условия.

Пример: 

age = 25
is_student = True

if age < 18:
    print("Вы несовершеннолетний.")
else:
if is_student:
    print("Вы взрослый студент.")
else:
    print("Вы взрослый, но не студент.")

Задача 1: Напишите программу, которая запрашивает у пользователя число и выводит сообщение "Число положительное", если число больше нуля, и "Число неположительное", если число меньше или равно нулю.

Задача 2: Напишите программу, которая запрашивает у пользователя оценку (от 0 до 100) и выводит текстовую интерпретацию оценки: - "Отлично!", если оценка 90 и выше, - "Хорошо!", если оценка от 75 до 89, - "Удовлетворительно!", если оценка от 60 до 74, - "Неудовлетворительно!", если оценка ниже 60.

Задача 3: Напишите программу, которая запрашивает у пользователя возраст и выводит сообщение: - "Вы ребенок", если возраст меньше 12 лет, - "Вы подросток", если возраст от 12 до 17 лет, - "Вы взрослый", если возраст 18 лет и старше.

Задача 4: Напишите программу, которая запрашивает у пользователя число от 1 до 7 и выводит название дня недели, соответствующее этому числу (1 - Понедельник, 2 - Вторник и т.д.). Если введенное число не входит в диапазон от 1 до 7, выводите сообщение "Некорректный ввод".

Задача 5: Напишите программу, которая запрашивает у пользователя два числа и выводит сообщение: - "Первое число больше второго", если первое число больше второго, - "Второе число больше первого", если второе число больше первого, - "Они равны", если оба числа равны. Также программа должна учитывать, если одно из чисел отрицательное, и выводить "Одно или оба числа отрицательные", если это так.
